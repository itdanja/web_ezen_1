요리

요리(料理, Cooking)는 먹기 좋게 가공한 음식이나 가공 행위 자체를 의미한다. 식재료를 가공하는 행위에는 '조리(調理)'도 명사로 쓰이지만, 음식을 보고 조리라고 부르지는 않는다 문단 안에 모델
